# Análisis de Datos de Ventas
Objetivo
Analizar un conjunto de datos de ventas para identificar patrones, evaluar la rentabilidad y generar insights que apoyen la toma de decisiones.

Descripción del dataset
El dataset contiene información sobre:
Pedidos (Order ID, fechas de compra y envío)
Clientes (Customer ID, nombre, segmento)
Ubicación (país, estado, región)
Productos (categoría, subcategoría, nombre)
Métricas (ventas, cantidad, descuento, ganancia)

Herramientas utilizadas
Excel (limpieza de datos y análisis)
Tablas dinámicas
Fórmulas

Limpieza de datos
Eliminación de duplicados
Estandarización de texto (ESPACIOS, LIMPIAR, NOMPROPIO)
Conversión de formatos (fechas y números)
Creación de variables:
shipping_date (días de envío)
discount_type (Alto/Bajo)

Análisis realizados
Ventas y rentabilidad
Cálculo de ventas totales y ganancias
Evaluación del margen de ganancia
Productos más vendidos
Ranking de productos por ventas
Identificación de Top 10 productos por región
Análisis geográfico
Ventas por estado
Análisis de clientes
Identificación de clientes con mayor volumen de compras
Logística
Tiempo promedio de envío por tipo de envío
Impacto de descuentos
Comparación de rentabilidad entre descuentos altos y bajos

Insights clave
Los descuentos altos generan pérdidas promedio (-97.18), mientras que los bajos generan ganancias (49.03)
Algunas regiones presentan altas ventas pero baja rentabilidad
El tiempo de envío varía significativamente según el tipo de envío
📸 Dashboard

Conclusión

El análisis permitió identificar factores que afectan la rentabilidad, destacando el impacto negativo de descuentos altos y la importancia de optimizar estrategias de ventas.
